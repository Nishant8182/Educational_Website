# RAISE | Educational Platform

RAISE is a modern, interactive educational platform designed to bridge the gap between curiosity and knowledge. It provides high-quality preparation materials for competitive exams and comprehensive computer science courses in a sleek, user-friendly environment.

## ✨ Key Features

-   **🌓 Dynamic Theme Toggling**: Seamlessly switch between Light and Dark modes with a persistent system that remembers your preference.
-   **🚀 Interactive Revealing Animations**: Modern scroll-based animations powered by the `Intersection Observer API` for an engaging user experience.
-   **📱 Fully Responsive**: Optimized for all devices, from desktop monitors to mobile phones.
-   **📚 Extensive Course Modules**:
    -   **JEE & GATE Preparation**: Specialized notes, mock tests, and video lectures.
    -   **Computer Science**: Master Java, Python, C++, Web Development, and Data Structures.
-   **🛡️ Premium Login System**: A dedicated, aesthetic landing page for student authentication and registration.
-   **🧩 Glassmorphism UI**: Uses modern design trends like backdrop blur and subtle gradients for a premium feel.

## 🛠️ Tech Stack

-   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
-   **Libraries**: [jQuery](https://jquery.com/) (Legacy support), [FontAwesome](https://fontawesome.com/) (Icons).
-   **Typography**: [Google Fonts](https://fonts.google.com/) (Outfit & Plus Jakarta Sans).
-   **Hosting Compatibility**: Ready for deployment on GitHub Pages or any static site host.

## 🚀 How to Run Locally

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Nishant8182/Educational_Website.git
    ```
2.  **Open in Browser**:
    Simply open the `index.html` file in your favorite web browser (Chrome, Firefox, Edge, etc.).
3.  **Development Server (Recommended)**:
    If using VS Code, use the **Live Server** extension for the best experience with real-time updates.
